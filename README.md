# sui-token-gen-test

Build
```
cargo run build
```

Create Command
```
cargo run create   
```

Verify Command

URL
```
cargo run verify --url https://github.com/meumar-osec/test-sui-token
```

Path
```
cargo run verify --path /Users/developer/Desktop/sui/sui-token
```

Tests
```
cargo test
```
